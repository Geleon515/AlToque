# AlToque
Plataforma web diseñada como un marketplace descentralizado de servicios técnicos y domésticos de oficio (como gasfitería, electricidad, carpintería, entre otros). El sistema conecta de manera eficiente y en tiempo real a clientes que tienen una necesidad inmediata en el hogar con técnicos calificados de su zona.
